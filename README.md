# QuickRunner

QuickRunner 是一款基于开源测试框架 HttpRunner 打造的简单易用、功能强大、数据精准、信创兼容的新一代API测试工具，将同时支持接口测试与性能测试。本次开放内测的是v1.0.12版本，已支持httprunner的大部分能力，可以作为一款完整的性能测试工具使用，欢迎体验试用和反馈~
